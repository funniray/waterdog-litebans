## Litebans-Waterdog  
  
Waterdog plugin that's similar to litebans.  
All of this was written 100% from scratch out of pure boredom.  
  
### Building:  
In a command line, run
```bash
mvn package
```  
The file will be `./target/litebans-waterdodge-X.X-SNAPSHOT.jar`
